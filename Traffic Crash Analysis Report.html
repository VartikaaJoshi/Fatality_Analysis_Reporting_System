Model Development & Prediction

A linear regression model was developed to predict the number of fatalities based on

various features. The RMSE achieved for this model was approximately 0.3441, which indicates the average error in predicting the number of fatalities.


Scalability, Scope, and Recommendations

As the dataset grows or as more features are added, the choice of model and the approach to data analysis may need to be revisited. Gathering more recent data, trying different models, and continuous model retraining are some of the recommendations for future work.
